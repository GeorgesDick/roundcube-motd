Bienvenue dans la nouvelle interface Webmail de MonacoNet !

Nous vous recommandons :
D'activer le rappel de mot de passe :
1) Préférences -> Identités -> bouton '+' en bas à gauche pour déclarer l'adresse qui recevra le mot de passe en cas d'oubli
2) Préférences -> Administration du compte -> Rappel du mot de passe pour mettre en place le rappel.
D'activer le carnet d'adresses automatique :
Préférences -> Carnet d'adresses -> case "Utiliser le carnet d'adresses automatique" -> bouton "Sauvegarder".

Pour continuer cliquez sur "Courriel" (en haut à droite)
